# Don't Lose Your Head

A competitive word game combining Hangman's letter-guessing with Battleship's grid-based hidden information.

## About

Players place three hidden words on grids and take turns guessing letters or coordinates to find their opponent's words before the guillotine blade falls.

## Tech Stack

- **Unity Version:** 6.2 (2D Template)
- **Platform:** PC/Mobile
- **Key Assets:**
  - DOTween Pro (animations)
  - Odin Inspector (data structures)
  - Easy Save 3 (save/load)
  - UGUI Super ScrollView
  - UI Assistant

## Core Mechanics

- Place 3 words (3, 4, and 5 letters) on a hidden grid
- Guess letters or specific coordinates
- Three difficulty levels (6x6, 8x8, 10x10 grids)
- Progressive miss counter with guillotine animation
- First to reveal all opponent's words wins
